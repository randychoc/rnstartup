# Next.js 

This is a Next.js with shadcn/ui.

## Adding components

To add components in the app, run the following command:

```bash
npx shadcn@latest add button
```

This will place the ui components in the `components` directory.

## Using components

To use the components in the app, import them as follows:

```tsx
import { Button } from "@/components/ui/button";
```
